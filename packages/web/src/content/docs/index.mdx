## Quick Start

```bash
# Install Echo AI
npm install -g echo-ai

# Start your first chat
echo chat "Help me write a Python function to parse JSON"

# Use with specific models
echo chat --model gpt-4 "Review this code for security issues"

# Interactive mode
echo tui
```
